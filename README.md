# Study Planner App

An AI-powered academic planning tool for college students that helps manage assignments, deadlines, study schedules, and productivity.

# Project Description

Study Planner App helps college students stay organized by tracking assignments, managing deadlines, creating personalized study schedules, and providing AI-powered study recommendations.

The application is designed to reduce stress, improve time management, and help students achieve better academic performance.

---

# Key Features

- Assignment and deadline tracking
- Personalized study schedules
- Calendar integration
- AI-powered study recommendations
- Focus Timer for productivity
- Progress tracking and analytics
- Assignment priority management
- Reminder notifications
- User authentication and account management

# End User

The primary users of the application are college students who struggle with:

- Managing assignments
- Keeping track of deadlines
- Creating study schedules
- Balancing school and personal responsibilities
- Improving productivity

---

# Problem Statement

Many college students have multiple classes, jobs, and extracurricular activities, making it difficult to stay organized.

Poor organization can lead to:

- Missed assignments
- Missed deadlines
- Lower grades
- Increased stress
- Poor time management

The Study Planner App addresses these challenges by providing a centralized platform for academic planning and productivity.
